## 博客


这个一个用`hexo`搭建的博客，用的主题是`next`。

- 把一周两周的积累的文章之类的知识，整理好一篇博客。
- 把日常工作的经验整理好一篇博客。


### 日常使用

```nginx
在source的_posts文件夹中，新建文章。

# 生成静态文件
$ hexo generate

# 启动服务器 并且端口为5000
$ hexo server -p5000

# 部署网站 
$ hexo deploy
```
